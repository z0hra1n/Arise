# Arise
This is arise, my first short parkour 3d godot game.It was my first game ever so i decided to make it as short and simple as possible, while covering the basics so i can get a strong foundation on working for future projects.

ai use in this project was very minimal i have experience with game dev with unity so i did have a rough idea on where to start even with godot.The coding system is so simple i didnt even need to use ai for code 99 percent of the code was written by me the remaining one percent was when i ran into serious issues and could not fix them myself so i used ai for that.Other than that, the game is fully made by me.

THe .exe file is compressed as a 7z file.
